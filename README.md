# busanmade

부산 맛집 · 카페 · 술집 순위 사이트 (프로토타입)

## 1. 사이트 열어보기

`index.html` 파일을 더블클릭하면 크롬에서 바로 열립니다.

- **홈** (`index.html`) — 그룹별로 카테고리 카드, 카테고리당 TOP 10
- **더보기** (`category.html`) — 카테고리당 TOP 20
- **상세** (`detail.html`) — 가게 정보 (주소·전화·영업시간·주차·메뉴·설명·네이버지도 링크)
- **관리자** (`admin.html`) — 가게 추가/수정/삭제

## 2. 관리자 비밀번호

기본값: `busan2026`

변경 방법: `js/data.js` 파일을 메모장으로 열어 `BM_ADMIN_PASSWORD` 줄의 값을 원하는 비밀번호로 바꾸고 저장.

## 3. 데이터는 어디에 저장되나요?

지금은 **브라우저 안(localStorage)** 에 저장됩니다.
즉, 본인 컴퓨터의 크롬에서 입력한 가게 정보는 그 컴퓨터에서만 보입니다.

다른 사람도 같이 보려면 다음 단계(서버 + DB)로 옮겨야 합니다. (4번 참고)

### 백업하기 / 옮기기
관리자 페이지의 **데이터 내보내기** 버튼 → JSON 파일 저장.
다른 PC/브라우저에서 **데이터 가져오기**로 그 파일을 올리면 그대로 복원됩니다.

## 4. 다음 단계 (실제 배포)

지금은 **디자인·흐름 확인용 프로토타입**입니다. 누구나 같은 데이터를 보게 하려면:

1. **백엔드 + DB** 추가 — 추천: Supabase 또는 Firebase (무료 시작 가능, 코딩 최소화)
2. **호스팅** — Vercel 또는 Netlify 무료 플랜
3. **도메인 연결** — `busanmade.com` 같은 도메인 구입 후 연결
4. **네이버 노출** — 네이버 서치어드바이저에 사이트 등록
5. **크롬에서 접속 가능** — 일반 웹사이트는 자동으로 크롬에서 열립니다 (별도 작업 불필요)

위 단계는 다음 대화에서 함께 진행할 수 있습니다.

## 5. 폴더 구조

```
web-s/
├─ index.html        홈 (TOP 10)
├─ category.html     카테고리 더보기 (TOP 20)
├─ detail.html       가게 상세
├─ admin.html        관리자
├─ css/
│   └─ style.css
└─ js/
    ├─ data.js       데이터 저장/로드 + 비밀번호
    ├─ main.js       홈 렌더링
    ├─ category.js   카테고리 페이지 렌더링
    ├─ detail.js     상세 페이지 렌더링
    └─ admin.js      관리자 기능
```
